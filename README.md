**YouTube Converter (MP3 / MP4)**

A simple YouTube to MP3 / MP4 converter written in Python with a graphical interface.
I created this project because I wanted a local tool instead of relying on online converter websites.

**Features**
Download YouTube videos as MP4
Download YouTube audio as MP3
Supports single videos and playlists
Simple and fast GUI
No installation required (portable .exe)
